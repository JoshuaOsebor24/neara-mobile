# Neara Mobile

Clean Expo Router foundation for a mobile app.

## Structure

- `app/` route tree with root stack and tab navigation
- `components/` shared UI building blocks
- `constants/` theme tokens
- `hooks/` app hooks
- `services/` frontend service layer

## Development

```bash
npm start
```

Tabs included:

- Home
- Chats
- Saved
- Profile
